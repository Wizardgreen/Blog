-CSS Grid
這次最大的收穫了！ Grid 算是比較嶄新的 CSS 屬性，與以往的 Float 排版或 Flex 不同，Grid 用了有點類似二元座標的方式進行繪製，特別適合用在向本週這樣排版很不規則的設計，可是從 Can I Use ? 可以看到有些瀏覽器還不能完全支援(對 IE 我就是在說你)，但其便利性絕對是指日可待。除此之外也附上一些本週學習Grid所讀的文章，希望能幫上想深入 Grid 的朋友們！

  • A Complete Guide to Grid - CSS Tricks
